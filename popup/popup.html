How to use LC & GFG Sync

1
Generate GitHub Token
Go to GitHub Settings > Developer Settings > Personal access tokens (classic). Generate a new token with the repo scope ticked.
2
Link Your Repo
In the Settings tab, paste your token and type your repository name (e.g. username/leetcode-solutions). Click "Save & Test". It will automatically create a private repo for you!
3
Solve & Submit
Head over to LeetCode or GeeksforGeeks and solve a problem. Once you click Submit and get an "Accepted" result, your code is automatically pushed to GitHub!
Tip: Want to sync past submissions? Use the "Sync All Past Submissions" button in the Settings tab!
